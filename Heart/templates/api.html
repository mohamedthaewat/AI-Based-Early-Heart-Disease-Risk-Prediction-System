📋 Available Endpoints
POST
/predict
Predict CVD risk score based on patient parameters
Request Body:
{ age: 54, sex: 1, trestbps: 130, chol: 223, thalch: 140, oldpeak: 0.5, ca: 0, cp: 0, thal: 3, exang: 0, fbs: 0, restecg: 1, slope: 2 }
Response:
{ risk: 45.3, level: "Moderate Risk", color: "orange", features: [...] }
POST
/chatbot
Chat with AI assistant about cardiovascular health
Request Body:
{ message: "Why is my risk high?", patient: { /* patient parameters */ } }
Response:
{ reply: "Your risk is high due to..." }
POST
/simulate
Simulate lifestyle changes and see risk reduction
Request Body:
{ scenario: "bp", /* 'bp', 'chol', 'active', 'oldpeak' */ age: 54, ...other_params }
GET
/api/dashboard
Get dashboard statistics and analytics
Response:
{ total_patients: 42, high_risk_count: 8, avg_risk_score: 43.2, model_accuracy: 84.5, top_factors: [...] }
GET
/api/model-performance
Get model performance metrics and feature importance
POST
/api/recommendations
Get personalized health recommendations
Response Includes:
lifestyle, diet, medication, monitoring recommendations
GET
/get_patients
Get list of all patients (with optional search)
Query Parameters:
search: "patient name" (optional)
